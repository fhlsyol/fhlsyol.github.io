SQL基础 1

sql又叫结构化查询语言，分为三大类：DDL(数据定义语言)、DML(数据操作语言)、DCL(数据控制语言)

  • DDL：

    create    (创建数据库和表)
drop    (删除数据库和表)
alter    (修改表)

  • DML：

    insert    (增)
delete    (删)
update    (改)
select    (查)

  • DCL：

    grant    (授权)
revoke    (召回授权)

  • 注意：

      -- sql的注释是再注释前加--
  -- sql中sql的关键字不区分大小写
  -- sqli语句每句结束后都要加分号！！！

一. DDL (数据定义语言)

1. 数据库的操作

1）创建数据库

  • create database 数据库名称;

    创建指定数据库，如果这个数据库已经存在，会报错

  • create database if not exists 数据库名称;

    当指定数据库不存在的时候，创建指定数据库

  • create database if not exists 数据库名称 default charset utf8;

    创建数据库的时候设置字符集编码方式为utf-8，让数据库支持中文数据的存储

    注意：这里要写utf8，而不是utf-8！！！

2）删除数据库

  • drop database 数据库名称;

    删除指定数据库，若不存在则报错

  • drop database if exists 数据库名称;

    当指定数据库存在时，删除指定数据库

3）切换 / 使用数据库

  • use 数据库;

    切换 / 使用指定数据库；切换后所有数据库相关操作都是针对这个数据库

B. 表的操作

1）创建表(数据库文件是通过表来存储数据的)

  • creat table 表名;

    creat table if not exists 表名(字段名1 类型名1 约束1, 字段名2 类型名2 约束2, ...);

    注意：

    
-- 表名：开发者自己命名，见名知意，一般需要加前缀t_/tb_表示表

-- 字段名：开发者自己命名，见名知意；
    -- 注意：字段一定要有一个字段来作为主键(可以表示唯一一条记录)；
    -- 主键不能为空，并且是唯一的，并且数据类型是整型

-- 类型名：必须是当前数据库支持的数据类型
    -- MySQL中常用的数据类型：
    -- int                (整型)
    -- float            (浮点型)
    -- varchar            (不定长字符串)
    -- text                (定长字符串)
    -- bit                (布尔)
    -- date/datetime    (日期和时间)
    -- ...

-- 约束：
    -- not null            (非空约束)
    -- unique            (唯一约束)
    -- default            (默认值约束)
    -- primary key        (主键约束)
    -- foreign            (外键约束)
    -- auto_increment    (自动增长)
    -- ...
    -- 约束可以没有，也可以有多个

  • 栗子：

      CREATE DATABASE if not EXISTS school DEFAULT charset utf8;
  USE school;
  CREATE TABLE IF NOT EXISTS tb_stuent
  (
  stuid INT PRIMARY KEY AUTO_INCREMENT,     -- 主键也可以如下设置
  stuname VARCHAR(20) NOT NULL,
  stubirth DATE,
  stugender BIT DEFAULT 1,
  stutel VARCHAR(11) UNIQUE,
  -- PRIMARY KEY (stuid)    -- 设置主键的另外一种方式
  );
2）删除表

drop table if exists 表名;

删除指定表

3） 修改表

  • 添加字段

    alter table 表名 add column 字段名 字段类型 字段约束;

    添加多个可以如下：

    ALTER TABLE tb_stuent ADD COLUMN 
(stuaddr VARCHAR(200),
stutest INT
);

  • 删除字段

    alter table 表名 drop column 字段名;

  • 修改字段名

    alter table 表名 change 原字段 新字段 新的类型;

二. DML (数据操作语言)

增删改查

1. 增

  • insert into 表名 values(值1, 值2, 值3, ...);

    按照表中字段的顺序依次给每个字段添加值

    -- 补充：

    -- 字符串            用引号引起来

    -- 数字             直接写

    -- MySQL中的日期(date)
        -- 日期字符串    (如'2019-11-11')
        -- 当前日期         date(now())
        -- 当前年          year(now())
        -- 当前月          month(now())
        -- ...

    -- bit(布尔)        1或0

  • insert into 表名 (字段名1, 字段名2, ...) values(值1, 值2, ...)；

    以指定的顺序给指定的字段添加值

  • insert into 表名 (字段名1, 字段名2, ...) values (值11, 值12, ...), (值21, 值22, ...), ...;

    以指定的顺序给指定的字段同时添加多个值

2. 删

  • delete from 表名;

    删除当前表中的所有记录 (清空表)

  • delete from 表名 where 条件;

    删除所有满足条件的记录

    -- sql的条件语句的写法(筛选)：

    -- 比较运算：
        -- =                (等于)
        -- <>                (不等于)
        -- <, >, <=, >=     (同数学)

    -- 逻辑运算：
        -- and                (逻辑与)
        -- or                (逻辑或)
        -- not                (逻辑非)

    -- 集合运算：
        -- in                (包含)

    -- 区间判断：
        -- between A and b    (A到B之间)

    -- 空值判断：
        -- is null            (空)
        -- is not null        (非空)

    -- 筛选判断(针对字符串)：
        -- like ...

    栗子：

    delete from 表名 where id>1014 and gendere=0;
-- 删除id大于1014的，性别为0的所有记录

delete from 表名 where name in ('李云龙', '田雨', '秀芹');
-- 删除名字是集合里的元素的所有记录(相当于or语句的简单写法)

delete from 表名 where birth between '1960-01-01' and '1980-12-31';
-- 删除出生日期是这两个时间之间的所有记录

delete from 表名 where tel is null;
-- 删除电话为空的所有记录

delete from 表名 where name like '张%'
-- 删除姓张的所有记录
-- % 通配符，表示任意个任意字符
-- _ 通配符，表示单个任意字符
3. 改

  • update 表名 set 字段1=新值1, 字段2=新值2, ...;

    将指定表中 所有记录 中指定的字段修改成指定的值

  • update 表名 set 字段1=新值1, 字段2=新值2, ... where 条件;

    将 满足条件的记录 的指定字段设置为指定的值

4. 查

  • select * from 表名

    获取指定表中所有的记录中的所有字段的数据

  • select 字段1, 字段2, 字段3, ... from 表名;

    映射

    获取表中所有记录的指定字段的数据

  • select 字段名1 as '新字段名1', 字段2, 字段3 from 表名;

    列重名

    as可以省略

    获取指定字段的数据同时，对取出来的数据的指定列重命名

  • 结果重新赋值 (主要针对布尔)

    • select if(字段, 值1, 值2) from 表名

      如果if中对应的字段的值是1，则取出来重新赋值为值1，否则赋值为值2

      注意：这样的写法是MySQL特有的写法

      栗子：

      select name, if(gender, '男', '女') as '性别' from 表名;
-- 获取姓名和性别，性别是1的重新赋值为'男'

  • select case 字段 when 值 then 新值1 else 新值2 end

    这是sql的通用写法

    栗子：

    select name, case stugender when 1 then '男' else '女' end as '性别' from 表名;
-- 获取姓名和性别，性别是1的重新赋值为'男'

  • 列合并 (查询的时候将多个字段合并成一个数据返回结果)

    select concat(字段1, 字段2, ...) from 表名;

    select concat(name, id) as '姓名学号' from 表名;
-- 将姓名和学号合并成一列，并重命名

  • 筛选

    上面的查询语法的后面都可以加where 条件对记录进行筛选

    如： select * from 表名 where 条件;

    栗子：

    select name, tel from 表名 where gender=0
-- 获取所有女生的姓名和电话号码

  • 对查询结果进行排序

    在查询语法的后面加order by 字段;

    将查询结果按指定字段的值从小到大进行排序

    select * from 表名 order by birth;
-- 按出生日期从早到晚排序

select * from 表名 where gender=1 order by birth;
-- 把所有的男生按出生日期从早到晚排序

order by 字段 asc;

从小到大排序，可以省略 (默认排序方式)

order by 字段 desc;

从大到小排序

多排序：

按顺序写多个字段即可

栗子：

  select * from 表名 order by gender, birth desc;
  -- 先按性别排序，然后性别相同的记录当中再按生日排序
