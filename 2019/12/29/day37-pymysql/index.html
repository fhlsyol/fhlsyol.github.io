pymysql基础

1. 安装pymysql

pip install pymysql

可以加参数 -i 下载源地址 从指定下载源下载

2. 连接MySQL

import  pymysql

连接对象 = pymysql.connect(  # 和指定MySQL建立连接，并且返回一个连接对象
    host='localhost',  # mysql主机地址
    port=3306,  # mysql服务器端口，一般3306
    user='root',  # mysql用户
    password='123456',  # 用户对应的密码, 可省
    database='school',  # 建立连接后默认操作的数据库，可省
    charset='utf8',  # 设置连接的数据库文件的编码方式
    autocommit=True,  # 是否自动提交
)

3. 通过连接获取游标对象

with 连接对象.cursor(查询返回值类型=None) as 游标对象:
    数据库操作上下文

查询返回值类型：
    None  -  查询结果以元组的形式返回
    pymysql.cursors.DictCursor  -  查询结果以字典的形式返回

数据库上下文：
    游标对象(数据库相关操作)，只有在上下文的缩进里才有效 (相当于文件操作的with open)

4. 执行SQL语句

cursor.execute('SQL语句')

5. 关闭连接

连接对象.close()

6. 操作实例

由于在python中，SQL语句是通过 字符串 利用游标的对象方法传递给数据库的，而python中的字符串操作多种多样，多姿多彩，所以可以利用python的字符串操作来生成需要的SQL语句

import pymysql


def query_table(connect):
    # 注意: 执行查询的sql语句,查询结果保存在游标对象中的
    sql_str = 'select * from tb_student;'
    with connect.cursor(pymysql.cursors.DictCursor) as cursor:
        result = cursor.execute(sql_str)
        print(result, cursor)

        # 注意: cursor中的查询结果，取一个就少一个
        # 1. 游标对象.fetchall() - 获取当前查询的所有的结果
        # all_result = cursor.fetchall()
        # print('查询结果的个数:', len(all_result))
        # for dic in all_result:
        #     print(dic['stuname'])
        #
        # all_result2 = cursor.fetchall()
        # print(all_result2)

        # 2. 游标对象.fetchone()  - 获取当前查询中一条数据
        print(cursor.fetchone())
        print(cursor.fetchone())

        # 3. 游标对象.fetchmany(size)   -- 获取当前查询中指定条数的数据
        print(cursor.fetchmany(2))
