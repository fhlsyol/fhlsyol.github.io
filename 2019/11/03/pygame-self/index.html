pygame简单说明

1. 游戏最小循环

import pygame

"""1. 初始化游戏"""
pygame.init()

"""2. 创建游戏窗口"""
# 1) 创建窗口
screen = pygame.display.set_mode((1200, 600))
# 2) 设置窗口颜色
screen.fill((180, 180, 180))
# 3) 设置窗口标题
pygame.display.set_caption('红红火火')
# 4) 第一次显示屏幕（第一次用flip, 以后在主循环中用update）
pygame.display.flip()

"""游戏主循环"""
while True:
    # 检测事件, 尽量再创建一个game_function.py, 在其中写函数
    gf.chet_event()
    # 刷新页面
    pygame.display.update()

2. 图片相关操作

1.，2.，3.略

4. 图片缩放和旋转

1) 缩放：pygame.transform.scale(缩放对象，目标大小) - 返回缩放后的图片对象

pic_tr1 = pygame.transform.scale(image, (80, 80))

2) 按比例缩放（带旋转）：pygame.transform.rotozoom(缩放对象，旋转角度，缩放比例)

pic_tr2 = pygame.transform.rotozoom(image, 90, 0.7)

3. 文字相关操作

1. 加载字体

pygame.font.Font(字体文件，字体大小) - 返回一个字体对象

font1 = pygame.font.Font('./img/dd.ttf', 20)

2. 根据字体对象创建文字对象

字体对象.render(文字内容，True，文字颜色，文字背景颜色 = None) - 返回一个类型是Surface

text = font1.render('hello', True, (0, 0, 255))

3. 显示文字对象

screen.blit(text, (100, 100))

4. 获取文字对象大小

w, h = text.get_size()

同图片缩放和旋转

4. 绘图

主要运用pygame.draw模块

1. 画线段

pygame.draw.line(画在哪, 线段的颜色, 起点坐标, 终点坐标, 线宽=1)

pygame.draw.line(screen, (255, 0, 0), (0, 20), (200, 20), 8)

画折线

pygame.draw.lines(画在哪, 颜色, 是否闭合, 点列表, 线宽=1)

points = [(10, 50), (80, 600), (70, 30), (40, 100)]
pygame.draw.lines(screen, (255, 0, 0), False, points, 5)

2. 画圆

pygame.draw.circle(画在哪, 线色, 圆心坐标, 半径, 线宽=0)

线宽=0的意思是填充，其他值的时候画出来是圆环

pygame.draw.circle(screen, (0, 0, 255), (100, 200), 80, 2)

3. 画矩形

pygame.draw.rect(画在哪, 颜色, 矩形范围, 线宽=0)

矩形范围：(左上角x坐标, 左上角y坐标, 宽度, 高度)

线宽=0的意思是填充

pygame.draw.rect(screen, (0, 0, 255), (10, 100, 200, 100), 5)

4. 画椭圆

pygame.draw.ellipse(画在哪, 颜色, 矩形范围, 线宽=0)

原理：矩形的内切圆，方法同画矩形

pygame.draw.ellipse(
    screen, (0, 0, 255), 
    (10, 100, 200, 100), 5)

5. 画弧线

原理：画椭圆的一段弧

pygame.draw.arc(画在哪, 颜色, 矩形范围, 起始弧度, 终止弧度, 线宽=1)

pygame.draw.arc(
    screen, 
    (0, 0, 255), 
    (10, 100, 200, 100), 
    0, pi/2, 4)

6. 画多边形

与画折线的区别在于此方法可填充

pygame.draw.polygon(画在哪, 颜色, 点列表, 线宽=0)

线宽=0的意思是填充

points = [(300, 300), (200, 400), (500, 100)]
pygame.draw.polygon(screen, (255, 0, 0), points, 3)

5. 事件

事件检测循环见最小循环

def chet_all_events():
    for event in pygame.event.get():
        if event.type == ... :
            ...

有事件发生才进入for循环

event的type值会根据事件类型的不同而不一样。

1. 点击关闭按钮时 –> 退出游戏

if event.type == pygame.QUIT:
    sys.exit()

2. 单击鼠标时 –>

elif event.type == pygame.MOUSEBUTTONDOWN:
    check_mouse_click_button()
    check_mouse_...()

def check_mouse_click_button():
    mouse_x, mouse_y = pygame.mouse.get_pos() # 定位鼠标单击位置
    """鼠标单击按钮时的响应"""
    if button1_s_player.rect.collidepoint(mouse_x, mouse_y):
       button1_s_player_action()
    ...
def check_mouse_...():
    mouse_x, mouse_y = pygame.mouse.get_pos() # 定位鼠标单击位置
    """鼠标单击...时的响应"""
    ...

当检测到鼠标操作时，event.pos是鼠标的坐标

get_pos()方法是获取事件发生时鼠标的位置

上面是按下鼠标时，还有其他的：

"""
MOUSEBUTTONUP  -  鼠标松开
MOUSEMOTION  -  鼠标移动
"""

3. 按下键盘时 –>

elif event.type == pygame.KEYDOWN:
    check_keydown_events(event)


def check_keydown_events(event):
    """响应按下按键"""
    if event.key == pygame.K_ESCAPE:  # 按下ESC键退出游戏
        sys.exit()
    elif ...

上面是按下键盘按键时，还有其他的：

"""
KEYUP  -  键盘弹起
"""

event.key是按下按键的ASSII码，用函数chr()可以将ASSII码转换为按键字母

也可以用pygame.K_xxx来判断是否按下xxx键

6. 音频播放

1. mixer_music模块

只能播放一个音频

pygame.mixer_music.play(循环次数) —-> 负1为无限循环

pygame.mixer_music.load('path')  # 加载音频文件
pygame.mixer_music.play()  # 播放音频文件
pygame.mixer_music.stop()  # 停止播放

2. Sound模块

能播放多个音频，返回一个播放器对象，只能播放音效，.wav文件

play(循环次数) 同上

sound1 = pygame.mixer.Sound('path1')  # 加载音频文件
sound2 = pygame.mixer.Sound('path2')

sound1.play()  # 播放音频
sound2.play()

sound1.stop()  # 关闭音效
sound2.stop()
