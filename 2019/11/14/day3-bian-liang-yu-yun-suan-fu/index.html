变量

1. 什么是变量

变量是用来申请空间存储数据的

1）声明变量的语法

变量名 = 值（弱语言中，直接赋值就是声明）

在其他强语言中：数据类型名 变量名 = 值

student_name = "小明"
age = 28
sum_num = 25 + 42
years_after_eighteen = age - 18
2）说明

变量名 - 程序猿自己命名的
要求：是标识符，不能是关键字
规范：见名知意；满足PEP8命名规范(所有字母都小写,下划线隔
开多个单词)；不能使用系统提供的函数名、类名、库名等
来给变量命名

= - 赋值符号，并不是等号，等号是==；将右边的数据赋值给左边的
变量

值 - 任何有结果的表达式；例如：数据、已经声明过的变量、运算表
达式等

2. 变量的使用

使用变量就是使用变量中的值，变量中的值可以修改

age = 18
print(age)
age = '你好'
print(age)

3. 同时声明多个变量

1）同时声明多个变量，赋相同的值

变量名1 = 变量名2 = 变量名3 = 值

2）同时声明多个变量，赋不同的值

变量名1， 变量名2， 变量名3 = 值1， 值2， 值3

4. Python声明变量的原理

变量三要素（python表达）：

a = 10

值 - 给变量赋值的数据（10）

类型 - 给变量赋值的数据的类型（int）

地址 - 给变量赋值的数据在内存中的地址，也就是变量中真正存储的东西（0xff）

1）强语言(Java/C等)：

类型名 变量名 = 值

int a = 10
a = 100

先用数据类型申请一定的内存空间，再往申请到的内存空间里存储数据。

重新赋值时，内存空间的地址不变。

变量给变量赋值时，重新申请内存空间。

比喻：

先在土地（内存）里挖个只能装特定物质的坑（申请到的内存空间），再将对应的该物质存储在坑里。

重新赋值时，将这个坑里的物质倒出，装入新的物质，坑还是原来的坑。

变量给变量赋值时，相当于把这个坑又重新挖了一个，物质重新填了一次。

2）弱语言(python等)

变量名 = 值

b = 10
b = 100

根据要存储的数据类型和大小，自动判断需要申请的空间的类型与大小，再申请内存空间。

重新赋值时，申请一个新的内存空间，将新的内存空间地址与变量名关联起来，并将旧的内存空间清空释放。地址会改变。

变量给变量赋值时，将原来变量的地址与新变量的变量名新增关联。

比喻：

先根据要存储的值规划挖的坑的大小与类型，再在土地（内存）里挖对应的适合与这个值的最节省土地（内存）的坑，再将值放进去。

重新赋值时，会根据要重新存储的值规划新的坑的大小和类型，再在土地（内存）上新挖一个适合与新值的坑，然后将新值放在新坑里，然后将坑的标志牌（变量名）移动到这个新的坑里，最后销毁旧坑。

变量给变量赋值时，相当于坑不变，给这个坑再立一个标志牌（标签）。

证明：

a = 10
print(a, type(a), id(a))
a = 100
print(a, type(a), id(a))
a = 'abc'
print(a, type(a), id(a))
b = a
print(b, type(b), id(b))

运算符

python支持的运算符：

数学运算符、比较运算符、逻辑运算符、赋值运算符、位运算符

1. 数学运算符

+、-、*、/、%、//、**

1）+、-、*、/ 同数学，略
2）% 取余（取模）

5 % 2 = 1

常用与判断奇偶、是否整除、取低位数（如获取一个整数的个位数）

3）// 整除（商只取整数部分）（向下取整）

负数整除除不尽的时候向下取整

5 // 2 = 2

-5 // 2 = -3

常用于取高位数（如获取一个整数的最高位）

4）** 幂运算

2 ** 10 = 1024

16 ** 0.5 = 4

2. 比较运算符（结果返回布尔值）

<、>、<=、>= 和数学一样

== 等于

!= 不等于

print(10 == 10)  # True
print(10 != 10)  # False

python支持连续比较：

age = 28
print(18 < age < 40)  # True

3. 逻辑运算符

and、or、not

逻辑与、或、非

逻辑运算的运算对象和结果一般都是布尔值

1）and

返回True的充要条件：均为True

True and True  ->  True
True and False  ->  False
False and True  ->  False
False and False  ->  False

使用场景：需要多个条件同时满足

if score >= 90 and sex = famle:
    print('优秀女学生')

随堂练习：判断一个数是否能同时被3和7整除

num = 35
print('能同时被3和7整除的命题结果: ', 
      num % 3 == 0 and num % 7 == 0)

结果是：

能同时被3和7整除的命题结果: False
2）or

返回False的充要条件：均为False

True or True -> True
False or True -> True
True or False -> True
False or False -> False

使用场景：多个条件中有一个满足即可

3）not

改变布尔值（True变False，False变True）

not True -> False
not False -> True
4）短路操作

and的短路操作：
a and b，a是False的话，后面的不会执行，直接False

def fuc1():
    print('函数被执行了')
    return True

False and fuc1()

上代码不会打印’函数被执行了’，若把False改为True，则会打印

or的短路操作：
a or b，a是True的话，后面的不会执行，直接True

def fuc1():
    print('函数被执行了')
    return True

True or fuc1()

上代码不会打印’函数被执行了’，若把True改为False，则会打印

4. 赋值运算符

=、+=、-=、=、/=、%=、//=、*=

要求：左边必须是变量；组合赋值运算符的左边必须是声明过的变量

组合赋值运算符是缩写，即

变量 += 值 -> 变量 = 变量 + 值

a += 1
a = a + 1

以上两种写法效果一样（前提是a已经声明过）

5. 运算符优先级

数学运算符 > 比较运算符 > 逻辑运算符 > 赋值运算符

数学运算符中同数学一样：幂运算 > 乘除 > 加减

6. 位运算符

所有的位运算都是针对数字的补码进行操作的

所有位都参加，不管是不是符号位

速度最快，直接从底层运算，不需要消耗额外的内存

位运算：

&（按位与），|（按位或），~（按位取反），^（按位异或）

<<（左移），>>（右移）

1）&（按位与）：

补码的每一个对应位置，均为1结果的对应位置才是1

1110 & 0111 = 0110
3 & 2
3的补码：00000011
2的补码：00000010
00000011 & 00000010 = 00000010 == 2

-7 & 3
-7的补码：10000111(原) -> 11111001(补)
3的补码：                 00000011
11111001 & 00000011 = 00000001 == 1

应用：高效判断数字的奇偶性
数字 & 1，结果是0则偶数，结果是1则奇数

2）|（按位或）

补码的每一个对应位置，均为0结果的对应位置才是0

1110 & 0111 = 1111
-7 | 3
-7的补码: 11111001
+3的补码：00000011
11111001 | 00000011 = 11111011
11111011 -> 11111010 -> 10000101 == -5
3）~（按位取反）

补码的每一个对应位置，0变1,1变0

~ 1101  -> 0010
~ -7
-7的补码: 11111001
~ 11111001 -> 00000110 == 6
4）^（按位异或）

补码的每一个对应位置，相同为0，不同为1

11011 ^ 10010 -> 01001
5）<<（左移）

补码的每一个对应位置左移一位

效果：数字 * 2**n

数字乘以2的n次方

2 << 1  ->  2 * 2 ** 1 == 4
3 << 2  ->  3 * 2 ** 2 == 12

用法：乘以2的n次方(算法常用)

6）>>（右移）

补码的每一个对应位置右移一位

效果：数字 // 2**n

数字整除2的n次方(除不尽时，取小，注意负数)

4 >> 1  ->  4 // 2 ** 1 = 2
9 >> 2  ->  9 // 2 ** 2 = 2
-9 >> 1  ->  -9 // 2 ** 1 = -5

分支结构

python代码默认情况下是从第一行开始，从上往下依次执行
分支结构：某段代码是否执行看条件

python中只有一种分支结构：if 语句

其他语言里一般有两种：if、switch

1. if 结构

1）语法：
if 条件语句：
    代码段
2）说明

if - 关键字，固定写法

条件语句 - 任何有结果的表示都可以

： - 固定写法（出现冒号的位置，后面一般会换行产生缩进）

代码段 - 和if保持一个缩进的语句（分支代码段，即满足条件才执行的）

3）执行过程

先判断条件语句的结果是否为True（如果条件语句的结果不是布尔值，就先转换为布尔值）

如果是True，则执行代码段，否则不执行

num = 10
if num & 1 == 0:
    print('偶数')

相当于： 如果…就…

2. if - else 结构

if 条件语句：
    代码段1
else:
    代码段2

满足条件执行代码段1，不满足执行代码段2

num = 10
if num & 1 == 0:
    print('偶数')
else:
    print('奇数')

相当于： 如果…就…否则…

"""
1. 写出判断一个数是否能同时被3和7整除的条件语句,
     并且打印对应的结果
"""

num1 = input('请输入整数： ')

try:  # 判断输入的数据是否可以转为整型，若不是，打印警告
    num1 = int(num1)
except ValueError:
    print('\n铁憨憨，让你输整数，你非要皮一下')
else:
    if num1 % 3 == 0 and num1 % 7 == 0:
        print('该数可以同时被3和7整除，商分别是：%d, %d'
              % (num1 / 3, num1 / 7))
    else:
        print('该数不能同时被3和7同时整除')
"""
2. 写出判断一个数是否能够被3或者7整除，
    但是不能同时被3和7整除，
    并且打印对应的结果
"""
num2 = input('请输入整数： ')

try:  # 判断输入的数据是否可以转为整型，若不是，打印警告
    num2 = int(num2)
except ValueError:
    print('\n铁憨憨，让你输整数，你非要皮一下')
else:
    check_3_or_7 = num2 % 3 == 0 or num2 % 7 == 0
    check_3_and_7 = num2 % 3 == 0 and num2 % 7 == 0
    if check_3_or_7 and not check_3_and_7:
        print('该数满足要求')
    else:
        print('该数不满足要求')
'''
3. 输入年，写代码判断输入的年是否是闰年，
    如果是输出'闰年',否则输出'不是闰年'
'''

while True:

    year = input('请输入公历年份(输入q退出)： ')

    if year == 'q':
        break
    else:

        try:  # 判断输入的数据是否可以转为整型，若不是，打印警告
            year = int(year)
            if year <= 0:
                raise ValueError
        except ValueError:
            print('\n铁憨憨，让你输公历年份，你非要皮一下，重来！\n')
        else:
            check_4 = year % 4 == 0
            check_100 = year % 100 == 0
            check_400 = year % 400 == 0
            if check_4:
                if not check_100:
                    print('闰年')
                else:
                    if check_400:
                        print('世纪闰年')
                    else:
                        print('不是闰年')
            else:
                print('不是闰年')
"""
4. 今天上课讲了15000秒的课，编程计算今天上课时间是多少小时，
     多少分钟，多少秒；以‘XX时XX分XX秒’的方式表示。
     例如: 时间67秒  —> 0时1分7秒
"""
time = 15000
s = time % 60
m = (time // 60) % 60
h = time // 60 // 60
print('\n%d时%d分%d秒\n' % (h, m, s))
"""
5. 定义两个变量保存一个人的身高和体重，
     编程实现判断这个人的身材是否正常!
     公式: 体重(kg)/身高(m)的平方值在18.5 ~ 24.9之间属于正常。
"""

while True:

    height = input('请输入身高(m)(输入q退出)： ')

    if height == 'q':
        break
    else:
        try:  # 判断输入的数据是否可以转为浮点型，若不是，打印警告
            height = float(height)
            if height <= 0:
                raise ValueError
        except ValueError:
            print('\n铁憨憨，让你输身高，你非要皮一下，重来！\n')
            continue

    weight = input('请输入体重(kg)(输入q退出)： ')

    if weight == 'q':
        break
    else:
        try:  # 判断输入的数据是否可以转为浮点型，若不是，打印警告
            weight = float(weight)
            if weight <= 0:
                raise ValueError
        except ValueError:
            print('\n铁憨憨，让你输体重，你非要皮一下，重来！\n')
            continue
        else:
            bmi = weight / height ** 2
            if bmi < 18.5:
                print('偏瘦')
            elif bmi <= 24.9:
                print('正常')
            elif bmi <= 29.9:
                print('偏胖')
            elif bmi <= 34.9:
                print('肥胖')
            elif bmi <= 39.9:
                print('重度肥胖')
            else:
                print('极重度肥胖')

# 4.输入年，写代码判断输入的年是否是闰年，如果是输出'闰年',否则输出'不是闰年'
# 闰年的条件: 1)能被4整除但是不能被100整除 2)能被400整除
year = int(input('请输入年份:'))
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print(year, '是闰年')
else:
    print(year, '不是闰年')


# 5.今天上课讲了15000秒的课，编程计算今天上课时间是多少小时，多少分钟，多少秒；以‘XX时XX分XX秒’的方式表示。
# 例如: 时间67秒 —> 0时1分7秒
time = 3601
h = time // (60*60)
m = time % (60*60) // 60
s = time % 60
print(h, "时", m, '分', s, '秒')

# 公式: 体重(kg)/身高(m)的平方值在18.5 ~ 24.9之间属于正常。

weight = 50
value = weight / height**2
print(value)
if 18.5 <= value <= 24.9:
    print('正常')
else:
