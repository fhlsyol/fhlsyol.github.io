分支结构2：

if - elif - else结构

部分内容转自 https://www.yiibai.com/python/python_if_else.html

elif语句允许检查多个表达式是否为True，并在其中一个条件求值为True时立即执行对应代码块。

与else语句类似，elif语句是可选的。 但是，与else语句不同的是，else语句最多可以有一个语句，在if之后可以有任意数量的elif语句。

语法

if expression1:
    statement(s)
elif expression2:
    statement(s)
elif expression3:
    statement(s)
else:
    statement(s)
Python

Python不像其他编程语言那样提供switch或case语句，但是可以使用if..elif...else语句来模拟switch case，如下所示：

示例

#!/usr/bin/python3

amount = int(input("Enter amount: "))

if amount<1000:
   discount = amount*0.05
   print ("Discount",discount)
elif amount<5000:
   discount = amount*0.10
   print ("Discount",discount)
else:
   discount = amount*0.15
   print ("Discount",discount)

print ("Net payable:",amount-discount)

3）else

循环自然死亡时，else的代码段会执行

循环强制死亡时，则不会

自然死亡：for循环把值取完；while条件为False

强制死亡：break

完整的for循环

for 变量 in 序列：
    循环体
else：
    代码段

完整的while循环

while 条件语句：
    循环体
else:
    代码段

自然死亡例子：

for x in range(4):
    print(x)
else:
    print('else')

def fibonacci(n):
    num_1 = 1
    num_2 = 1
    for i in range(0, n - 2):  # 也可以写num_1, num_2 = num_2, num_1 + num_2
        temp = num_1 + num_2
        num_1 = num_2
        num_2 = temp
    print(num_2)

def solution1(n):
    """方法1"""
    num_str = str(n)  # 核心思想：转化为字符串
    num_list = []

    # 将每一位数字按序存储在列表中
    for i in range(0, len(num_str)):

    # 打印数字位数
    print(len(num_list))

    # 反转列表，并打印
    num_list.reverse()
    for j in num_list:
        print(j, end='')

def reverse_triangle(n):
    """倒三角输出"""
    while n > 0:
        for i in range(0, n):
            print('*', end='')
        n -= 1
        print('')


def triangle(n):
    """居中正三角输出"""
    num_n = n
    while n > 0:
        space_num = int((n - 1) / 2)
        space_str = ' ' * space_num
        print(space_str, end='')
        star_num = num_n - space_num * 2
        star_str = '*' * star_num
        print(star_str, end='')
        print(space_str, end='')

        n -= 2
        print('')

# 小马最多： 100 * 2 == 200      -> z
for x in range(1, 34):
    for y in range(1, 51):
        for z in range(2, 101, 2):
            if x*3 + y*2 + z/2 == 100 and x + y + z == 100:
                print('大马:%d, 中马:%d, 小马:%d' % (x, y, z))
                break
